# SistemaCursos

Este projeto foi gerado utilizando a [Angular CLI](https://github.com/angular/angular-cli) na versão 21.2.12.

## Servidor de Desenvolvimento

Para iniciar um servidor de desenvolvimento local, execute o comando:

```bash
ng serve
```

Assim que o servidor estiver rodando, abra o seu navegador e acesse http://localhost:4200/. A aplicação será recarregada automaticamente sempre que você modificar qualquer um dos arquivos de origem.

## Geração de Código (Scaffolding)
A Angular CLI possui ferramentas poderosas de geração de código. Para criar um novo componente, execute:
##ng generate component nome-do-componente

Para ver a lista completa de esquemas disponíveis (como components, directives ou pipes), execute:
```bash
ng generate --help
```
## Compilação (Build)
Para compilar o projeto, execute:
```bash
ng build
```

Isso compilará o seu projeto e armazenará os artefatos gerados no diretório dist/. Por padrão, a compilação de produção otimiza a sua aplicação para obter o máximo de desempenho e velocidade.

## Executando Testes Unitários
Para executar os testes unitários com o executor de testes Vitest, utilize o seguinte comando:
```bash
ng test
````

## Executando Testes de Ponta a Ponta (E2E)
Para testes de ponta a ponta (e2e), execute:
```bash
ng e2e
```

A Angular CLI não vem com um framework de testes de ponta a ponta configurado por padrão. Você pode escolher e instalar o que melhor atenda às suas necessidades
